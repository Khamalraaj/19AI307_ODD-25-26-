## SOURCE CODE:

```
import java.util.*;

public class seo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String s1 = sc.nextLine();
        String s2 = sc.nextLine();

        String con = s1.concat(s2);

        System.out.println("Concatenated string: " + con);
    }
}
```
